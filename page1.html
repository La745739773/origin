C++ Socket网络编程
自学项目

基于C++ Socket API 和 select网络通信模型构建客户端和服务端程序。掌握使用C++进行TCP通信的方法，掌握select网络通信模型的构建方法，掌握多线程、内存管理（内存池、对象池）的实现和运用。在单线程环境下，服务端到客户端一对多的非阻塞运行。通过构建精准的计时器，网络数据收发量达到1Gb每秒，数据包传送个数约6万个每秒。在多线程环境下，对服务端和客户端的收发数据逻辑和select查询分别放入不同线程进行扩展，是客户端socket数量突破10000个连接上限。

互联网地图爬虫

基于互联网地图(高德、百度)web服务API获取地图导航结果和POI数据。引入多线程编程：设计生产者-消费者模式，并加入Python消息队列(from multiprocessing import Queue)和线程池(from multiprocessing import Pool)实现高并发获取地图服务。

新浪微博爬虫

基于Python + Selenium 对新浪微博(非主站)和中国大学慕课进行数据爬取，攻克新浪微博和中国大学慕课网站的发爬虫技术。新浪微博：模拟登陆，通过多个微博账号模拟登陆手机端url(https://weibo.cn/),并破解了站点的滑块验证码。
